Npm install in HiraganaQuiz

cd backend/
npm install

cd ../frontend/
npm install

Add keys and info for following .env variables
AZURE_OPENAI_API_VERSION=
AZURE_OPENAI_API_INSTANCE_NAME=
AZURE_OPENAI_API_KEY=
AZURE_OPENAI_API_DEPLOYMENT_NAME=
AZURE_OPENAI_API_EMBEDDINGS_DEPLOYMENT_NAME=

npm start in HiraganaQuiz to start both frontend and backend
